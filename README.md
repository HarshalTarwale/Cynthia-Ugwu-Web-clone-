# Product Designer Website Clone

A modern recreation of a premium product designer portfolio website, built with vanilla JavaScript and cutting-edge animation libraries. This project showcases advanced UI interactions, smooth scrolling mechanics, and sophisticated design to deliver a professional creative portfolio experience.

## Screenshots

![App Screenshot](https://github.com/HarshalTarwale/Cynthia-Ugwu-Web-clone-/blob/main/product-designer-ss.png)

## Tech Stack

* **HTML5:** Semantic markup structure for optimal accessibility and SEO.
* **CSS3:** Modern styling with custom properties, flexbox, and grid layouts.
* **Vanilla JavaScript:** Pure JavaScript for DOM manipulation and interactive functionality.
* **Locomotive Scroll:** Premium smooth scrolling library providing hardware-accelerated, buttery-smooth scroll behavior.
* **Custom Animations:** Hand-crafted CSS animations and JavaScript-driven interactions for fluid user experience.

## Sections

1. **Hero Section:** Large-scale typography with engaging animations, featuring compelling headline and call-to-action elements.

2. **Portfolio Showcase:** Grid-based project gallery displaying creative work with hover effects and detailed project information.

3. **About Section:** Professional biography and skills presentation, highlighting expertise and design philosophy.

4. **Services:** Comprehensive overview of offered services with clean, organized layout.

5. **Interactive Elements:** Dynamic components that respond to user interaction, creating an engaging browsing experience.

6. **Contact Section:** Clear call-to-action with contact information and connection options for potential clients.

7. **Footer:** Professional footer with navigation links, social media connections, and copyright information.
